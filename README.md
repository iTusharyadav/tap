## 🛠 **Installation**

### Requirements

Before you start, ensure you have the following:

- **Node.js** (v16+)
- **Yarn** or **npm**

### Setup

1. Clone the repo:

   ```bash
   git clone https://github.com/your-username/codecraft.git
   ```

2. Navigate to the project directory:

   ```bash
   cd codecraft
   ```

3. Install the dependencies:

   ```bash
   npm install  # or yarn install
   ```

4. Set up your **Convex** and **Clerk** keys in a `.env.local` file:

   ```bash
   NEXT_PUBLIC_CONVEX_API_KEY=your_convex_api_key
   CLERK_FRONTEND_API=your_clerk_frontend_api_key
   ```

5. Start the app:

   ```bash
   npm run dev  # or yarn dev
   ```

   Your app is now running at `http://localhost:3000`.
