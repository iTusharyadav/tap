# Code Editor Project 🚀

Welcome to the Code Editor project! This is an advanced online IDE built with Next.js, Convex, Clerk, and TypeScript. The editor supports multiple programming languages and comes with a customizable and professional deployment experience. Explore and build code with ease!

![Tech Stack GIF](URL-to-your-tech-stack-gif) 

## 🚀 Tech Stack

This project uses the following technologies:

- **Next.js 15**: A React framework for building dynamic, scalable web applications.
- **Convex**: Backend framework for seamless server-side operations and real-time data synchronization.
- **Clerk**: Authentication platform for user management.
- **TypeScript**: For type safety and enhanced development experience.

## 🌟 Features

- **Multi-Language Support**: Run code in 10 different programming languages.
- **Customizable Themes**: Choose from 5 VSCode-inspired themes to suit your workflow.
- **Smart Output Handling**: Distinguish between Success & Error states in output to enhance the debugging process.
- **Flexible Pricing**: Free & Pro plans to accommodate a wide range of users.
- **Community-driven Code Sharing**: Share your code with the community and discover others' projects.
- **Advanced Filtering & Search**: Effortlessly search through your codebase with advanced filtering options.
- **Personal Profile**: Keep track of your execution history and display your personal profile with achievements.
- **Statistics Dashboard**: Monitor your coding activity with detailed statistics.
- **Customizable Font Size**: Adjust the editor's font size to your preference.
- **Webhook Integration**: Seamlessly integrate your workflow with webhook support.
- **Professional Deployment Walkthrough**: Step-by-step instructions for deploying your projects.

## ⚡ Getting Started

To get started with the project locally, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/your-repo/code-editor.git
    cd code-editor
    ```

2. Install dependencies:

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3. Run the development server:

    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    # or
    bun dev
    ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the editor in action.

> **Tip**: The page auto-updates as you edit the code, so no need to refresh manually.

### Start Editing

You can begin editing the page by modifying `app/page.tsx`. The app will auto-update as you make changes.
